1. Create a Java program to access the java.util.Date class through import keyword and print the current date of the system.

    import java.util.Date;

    public class CurrentDate {
        public static void main(String[] args) {
            Date date = new Date();
            System.out.println(date);
        }
    }

Output: Mon Nov 11 09:30:00 IST 2024

2. Create a Java program to access the java.util.Date class through qualified way and print the current date of the system.

    public class CurrentDateQualified {
        public static void main(String[] args) {
            java.util.Date date = new java.util.Date();
            System.out.println(date);
        }
    }

Output: Mon Nov 11 09:30:00 IST 2024

6. Create a Java program to create a package with 4 public classes, create the object in another file and call the function created in those classes.

    // Package: mypackage
    package mypackage;

    public class ClassOne { public void display() { System.out.println("ClassOne method"); } }
    public class ClassTwo { public void display() { System.out.println("ClassTwo method"); } }
    public class ClassThree { public void display() { System.out.println("ClassThree method"); } }
    public class ClassFour { public void display() { System.out.println("ClassFour method"); } }

    // Another file
    import mypackage.*;

    public class Main {
        public static void main(String[] args) {
            new ClassOne().display();
            new ClassTwo().display();
            new ClassThree().display();
            new ClassFour().display();
        }
    }

Output: ClassOne method
ClassTwo method
ClassThree method
ClassFour method

8. Create a Java program to create a package with public class and protected members to be accessed in another class.

    // Package: mypackage
    package mypackage;

    public class PublicClass {
        protected void display() {
            System.out.println("Protected Display Method");
        }
    }

    // Another file
    import mypackage.PublicClass;

    public class Main extends PublicClass {
        public static void main(String[] args) {
            Main obj = new Main();
            obj.display();
        }
    }

Output: Protected Display Method
