1. Enter the character from user using InputStreamReader

    import java.io.*;

    public class CharacterInput {
        public static void main(String[] args) throws IOException {
            InputStreamReader reader = new InputStreamReader(System.in);
            System.out.print("Enter a character: ");
            char character = (char) reader.read();
            System.out.println("You entered: " + character);
        }
    }

Output: Enter a character: a
You entered: a

9. Get all characters from the file and display on the screen

    import java.io.*;

    public class FileReaderExample {
        public static void main(String[] args) throws IOException {
            FileReader fileReader = new FileReader("file.txt");
            int character;
            while ((character = fileReader.read()) != -1) {
                System.out.print((char) character);
            }
            fileReader.close();
        }
    }

Output: (This will display the content of the file 'file.txt' on the screen)

10. Get all characters from the file, count number of lines, words, characters and display on the screen

    import java.io.*;

    public class FileStatistics {
        public static void main(String[] args) throws IOException {
            BufferedReader reader = new BufferedReader(new FileReader("file.txt"));
            String line;
            int lineCount = 0, wordCount = 0, charCount = 0;
            
            while ((line = reader.readLine()) != null) {
                lineCount++;
                String[] words = line.split("\\s+");
                wordCount += words.length;
                charCount += line.length();
            }
            reader.close();
            
            System.out.println("Lines: " + lineCount);
            System.out.println("Words: " + wordCount);
            System.out.println("Characters: " + charCount);
        }
    }

Output: Lines: 5
Words: 15
Characters: 50 (Output will vary based on file content)
