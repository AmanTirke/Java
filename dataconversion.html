1. How to convert String to int

String str = "123";
int num = Integer.parseInt(str);
Output: 123

2. How to convert int to String

int num = 123;
String str = Integer.toString(num);
Output: "123"

3. How to convert String to long

String str = "123456789";
long num = Long.parseLong(str);
Output: 123456789

4. How to convert long to String

long num = 123456789L;
String str = Long.toString(num);
Output: "123456789"

9. How to convert String to Date

String str = "2024-11-11";
SimpleDateFormat format = new SimpleDateFormat("yyyy-MM-dd");
Date date = format.parse(str);
Output: Tue Nov 11 00:00:00 GMT 2024

10. How to convert Date to String

Date date = new Date();
SimpleDateFormat format = new SimpleDateFormat("yyyy-MM-dd");
String str = format.format(date);
Output: "2024-11-11" (example)
