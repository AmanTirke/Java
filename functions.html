1. Write a Java method to find the smallest number among three numbers.

Method:

Finds the smallest number among three provided numbers.

public static int findSmallest(int num1, int num2, int num3) {
    return Math.min(num1, Math.min(num2, num3));
}

Output:

Smallest number: 3

3. Write a Java method to display the middle character of a string.

Method:

Displays the middle character(s) of the given string.

public static String middleCharacter(String str) {
    int len = str.length();
    if (len % 2 == 0) {
        return String.valueOf(str.charAt(len / 2 - 1));
    } else {
        return str.substring(len / 2 - 1, len / 2 + 1);
    }
}

Output:

Middle character(s): "n"

5. Write a Java method to compute the future investment value at a given interest rate for a specified number of years.

Method:

Computes the future investment value based on principal, interest rate, and years.

public static double computeFutureInvestmentValue(double principal, double annualInterestRate, int years) {
    return principal * Math.pow(1 + annualInterestRate / 100, years);
}

Output:

6. Write a Java method to print characters between two characters (i.e. A to P).

Method:

Prints characters between the two given characters.

public static void printCharactersBetween(char start, char end) {
    for (char c = start; c <= end; c++) {
        System.out.print(c + " ");
    }
}

Output:

Characters: A B C D E F G H I J K L M N O P

7. Write a Java method to check whether a string is a valid password.

Method:

Checks if the provided string is a valid password (length >= 8, contains digits, uppercase and lowercase letters).

public static boolean isValidPassword(String password) {
    return password.length() >= 8 && password.matches(".*[0-9].*") && password.matches(".*[a-z].*") && password.matches(".*[A-Z].*");
}

Output:

Valid Password: true

9. Write a Java method to find all twin prime numbers less than 100.

Method:

Finds twin prime numbers less than 100.

public static void findTwinPrimes() {
    for (int i = 2; i < 100; i++) {
        if (isPrime(i) && isPrime(i + 2)) {
            System.out.println(i + " and " + (i + 2));
        }
    }
}

private static boolean isPrime(int num) {
    for (int i = 2; i <= Math.sqrt(num); i++) {
        if (num % i == 0) {
            return false;
        }
    }
    return true;
}

Output:

Twin Primes: 3 and 5, 5 and 7, 11 and 13, 17 and 19, 29 and 31, 41 and 43, 59 and 61, 71 and 73

13. Write a Java method that checks whether all the characters in a given string are vowels (a, e, i, o, u) or not. Return true if each character in the string is a vowel, otherwise return false.

Method:

Checks if each character in the string is a vowel.

public static boolean allVowels(String str) {
    for (char c : str.toLowerCase().toCharArray()) {
        if ("aeiou".indexOf(c) == -1) {
            return false;
        }
    }
    return true;
}

Output:

All Vowels: false
